# Imersão dados com Python da Alura

Projeto realizado com base nos conhecimentos adquiridos através da imersão dados com Python ofertado pela Alura em Fev/2026.
A base de dados foi disponibilizada através de um repositório do GitHub.

## Arquitetura

O projeto segue a seguinte arquitetura:

```

├── main.py/       # Código em Python para criação de página em Streamlit
├── main.ipynb/    # Código no Jupyter Notebook para limpeza e processamento dos dados
├── df_final.csv/  # DataFrame após limpeza
└── .streamlit/    # Configurações de style do Streamlit
